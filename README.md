## Getting Started

Clone this project and install dependencies :

npm i

# or

yarn install

Add your environment variables (nextAuth\NotionApi)
Runs the app in the development mode:

npm run start

# or

yarn start

Runs the app in the development mode.
Open http://localhost:3000 to view it in your browser.
